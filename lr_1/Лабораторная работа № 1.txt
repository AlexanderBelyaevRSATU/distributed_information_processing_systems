Лабораторная работа № 1
Тема: “Знакомство с Linux”




1. Изучение базовых команд консоли:
   1. Перемещение по каталогам файловой системы
   2. Создание/удаление каталогов и файлов
   3. Выделение прав для файлов (чтение, изменение,…)
   4. Просмотр содержимого каталога
   5. Монтирование дисков
   6. Изучение man
2. Работа с пользователями системы:
   1. Создание пользователя с помощью консоли
   2. Выделение прав для пользователя
   3. Удаление пользователя (прежде чем удалять показать что создали и зашли под этим пользователем)
3. Установка и удаление программ
   1. Изучить способы установки программ на linux
   2. Что такое snap, apt, apt-get
   3. Установить и удалить с помощью консоли ПО (например Sublime, если есть уже, то можно любое другое)
4. Shell, curl, SSH
   1. Изучить shell - скрипты
   2. Изучить curl и попробовать
   3. Написать скрипт (продемонстрировать его работу)
   4. Изучить ssh
